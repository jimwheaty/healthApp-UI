Scores


- WeightLossScore = (C + φυτικές ίνες + κάλιο + μαγνήσιο + MCT + ωμέγα-3 - 2xποσοστόΖάχαρης - 2xποσοστόΓλυκόζης - 2xνάτριο/νερό) / maxWeightLossScore

- EnergyScore = (B1 + B2 + B3 + B6 + B9 + B12 + E + ασβέστιο + χαλκός + σίδηρος + μαγνήσιο + φώσφορος + κάλιο + νάτριο) / maxEnergyScore

- HormoneScore = (ασβέστιο + σελήνιο + ψευδάργυρος) / maxHormoneScore

- SkinScore = (A + B3 + C + E + Omega-3) / maxSkinScore

- BrainScore = (B1 + B12 + χολίνη) / maxBrainScore

- MuscleScore = (A + B1 + B2 + B9 + χολίνη + ασβέστιο + σίδηρο + μαγνήσιο + φώσφορος + κάλιο + νάτριο) / maxMuscleScore



- GutScore = (φυτικές ίνες + omega-3 - 2xlongChainSaturatedFat) / maxGutScore

- HeartScore = (K + ασβέστιο + χαλκός + μαγνήσιο + κάλιο + ψευδάργυρος + omega-3) / maxHeartScore

- ImmuneScore = (B6 + E + χαλκός + μαγνήσιο + σελήνιο + ψευδάργυρος) / maxImmuneScore

- StressScore = (B1 + B3 + B6 + ασβέστιο + χαλκός + μαγνήσιο + κάλιο + νάτριο) / maxStressScore

- AntiAgingScore = (Α + C + E + φώσφορος + σελήνιο + ψευδάργυρος) / maxAntiAgingScore

-BonesTeethScore = (A + C + K + ασβέστιο + χαλκός + μαγνήσιο + φώσφορος) / maxBonesTeethScore

- GrowthScore = (A + B2 + B9 + C + φώσφορος + σελήνιο + ψευδάργυρος) / maxGrowthScore

- ThyroidScore = (νάτριο + σελήνιο + ψευδάργυρος) / maxThyroidScore

- EyeScore = (A + ψευδάργυρος + omega-3) / maxEyeScore
